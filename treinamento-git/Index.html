Prezad@s hoje teremos uma aula assíncrona com o objetivo de praticar o conteúdo visto, então no caso não teremos aula no meets.

Sua presença vai ser contabilizada com a entrega dessa atividade.
Se por algum motivo você não conseguir concluir, você deve entregar uma justificativa.
Instruções

  1. Criar um projeto novo na sua maquina como o nome "treinamento-git";
  2. Iniciar o git;
  3. Criar um arquivo index.html com a estrutura básica vista no exemplo;
  4. Fazer 5 modificações pergunta e resposta 1 commit para cada modificação;.
    • Quais os comandos para configurar usuário e email no git?
    • Qual a diferença entre git e github?
    • O que faz o comando commit?
    • O que faz o comando push?
    • O que faz o comando pull?
  5. Sincronizar com o guithub;
  6. Publicar no github pages
Itens de entrega
link - código fonte no github
link - da sua página html no github pages
