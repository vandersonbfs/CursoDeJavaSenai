Perguntas

Quais os comandos para configurar usuário e email no git?

git config --global user.email "vandersonb@gmail.com"

Qual a diferença entre git e github?

git é um software VCS local que permite aos desenvolvedores salvar snapshots de seus projetos ao longo do tempo.
Geralmente é melhor para uso individual. GitHub é uma plataforma baseada na web que incorpora os recursos de controle
de versões do gitHub para que possam ser usados colaborativamente.

O que faz o comando commit?

O comando git commit captura um instantâneo das mudanças preparadas do projeto no momento.
Os instantâneos com commit podem ser considerados versões "seguras" de um projeto,
o Git nunca os altera, a menos que você peça a ele.

O que faz o comando push?

O comando push transfere commits do repositório local a um repositório remoto.
É o oposto do comando git fetch , que importa commits para branches locais,
enquanto o comando push exporta commits para branches remotos.

O que faz o comando pull?

O comando git pull é usado para buscar e baixar conteúdo de repositórios remotos e fazer a atualização imediata
ao repositório local para que os conteúdos sejam iguais. Fazer o merge de alterações upstream remotas no repositório
local é algo comum em fluxos de trabalho de colaboração baseados em Git.
